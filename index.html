☁️ Sincronização na Nuvem — Firebase
☁️

Configure o Firebase gratuito do Google para sincronizar seus dados em tempo real entre PC, celular e tablet. Os dados ficam na sua própria conta Google — Anthropic não tem acesso a nada.

📋 Como configurar (5 minutos)
1
Acesse console.firebase.google.com e clique em "Criar um projeto". Dê um nome (ex: "mesas-trader") e desative o Google Analytics.
2
No menu lateral, clique em Criação → Firestore Database → "Criar banco de dados" → escolha "Modo de produção" → selecione a região southamerica-east1.
3
Ainda no Firestore, vá em Regras e substitua o conteúdo por:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
4
No menu lateral, clique em Criação → Authentication → "Primeiros passos" → ative o provedor Google.
5
Clique na engrenagem ⚙️ → Configurações do projeto → role até "Seus apps" → clique no ícone </> (Web) → registre o app → copie o objeto firebaseConfig e cole abaixo.
Cole o JSON exatamente como aparece no console do Firebase
